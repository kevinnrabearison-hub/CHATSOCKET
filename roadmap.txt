🏗 ROADMAP PROFESSIONNELLE DU PROJET

Je vais te donner un plan structuré comme un architecte backend.

PHASE 1 — AUTHENTIFICATION (OBLIGATOIRE)

Avant WebSocket, avant rooms, avant messages :

👉 On doit implémenter JWT.

Pourquoi ?

Parce que :

WebSocket doit être sécurisé

Les messages doivent être liés à un user authentifié

Les rooms doivent connaître l’utilisateur

🎯 Objectif phase 1

Créer :

/auth/login
/auth/register
JwtStrategy
JwtGuard

Résultat :

Login → retourne un token

Token → utilisé pour REST + WebSocket

Plusieurs instances Nest

Messages synchronisés

Scalabilité réelle
